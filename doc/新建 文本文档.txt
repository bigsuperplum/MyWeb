网页整体框架
该网页用于向用户推荐 5 个游戏网站，每个都有图片介绍，并且附有网站地
址，点击可以跳转。同时为了方便用户进行更深一步搜索，附有搜索框 (输入内
容，并点击搜索之后跳转必应的搜索结果)，还有哔哩哔哩网站的跳转按键。网
页整体框架如图1-1
1) CSS
2) Java
3) HTML
网页整体的搭建使用了如上三种技术：HTML、CSS 和 JavaScript。
HTML 部分：主要用于构建网页的结构和内容。例如 <!DOCTYPE html> 声
明文档类型和 HTML 版本；<html> 标签定义了 HTML 文档；<head> 标签包含
了文档的元数据，如 <title> 定义了页面标题；<body> 标签包含了可见的页面内
容，如 <h1>, <h2>, <p> 等标签。
CSS 部分：用于设置网页的样式。<style> 标签内定义了 CSS 样式规则，
如.container, .image, .search-box 和 body 的样式。使得内容居中等。
JavaScript 部分：用于网页的交互功能。<script> 标签内包含了 JavaScript 代
码，定义了一个名为 hello 的函数，当调用时会显示一个警告框。

主页设计
1) 欢迎区域：标题：在页面顶部中央，使用大号、醒目的字体显示“!! welcome!!”，
以吸引用户的注意力。图标：在标题下方居中放置一个可爱的华小科动漫
角色图标，增加网站亲和力。
2) 推荐游戏区域：标题：在图标下方，使用较大字体显示“推荐”，简洁明了
地告诉用户接下来是游戏推荐内容。游戏列表：列出五个推荐的游戏，每个
游戏使用不同的颜色或样式的按钮，以便用户快速识别。
3) 搜索和了解更多区域：搜索框：在推荐游戏下方，提供一个搜索框，用户可
以输入关键词进行搜索。搜索框旁边放置一个按钮，标注“让华小科搜索”，
点击后触发搜索功能。哔哩哔哩跳转：在搜索框下方，提供一个链接或按
钮，标注“点击去 bb 了解更多”，点击后跳转到哔哩哔哩网站，方便用户获
取更多游戏信息和攻略。
4) 设计细节：背景颜色：选择一种轻松愉快的浅蓝色，营造友好的浏览环境。
字体和排版：使用清晰易读的字体，确保信息层次分明，用户可以快速找到
所需内容。
5) 响应式设计：确保主页在不同设备上都能良好显示，包括桌面、平板和手
机。

网页设计小结
在设计网页过程中遇到了以下困难：
1) 如何设计一个美观，且能够达到推荐游戏的目的的界面
2) 如何兼具网页的丰富性和一致性
3) 如何确保网页在不同设备和屏幕尺寸上都能良好显示
4) 如何设计直观的用户界面
5) 如何提供快速的页面加载时间和流畅的交互
6) 如何吸引更多的访问者
解决方案如下：
1) 确定网页的核心功能和主要内容，美观方面：让所有内容居中显示。实用方
面：不仅提供简介，还未用户提供快捷跳转的方式，同时提供更深一步搜索
的快捷途径
2) 选择浅色的基调的同时，选用丰富的颜色
3) 进行多设备测试，确保兼容性
4) 学习简约的界面设计便于用户访问
5) 使用用户测试来优化交互设计
6) 丰富网页细节并提供良心的游戏网站推荐
在网页的设计和制作过程中主要出现了以上困难，本人通过学习和交流将遇到
的问题一一解决。最终让用户能够使用具有一定美观性，又实用、快捷的游戏推
荐网站。

课程的收获和建议
5.1
计算机基础知识
通过本课程我了解到了计算机，特别是有关网页制作的基本知识，学习计算
机基础知识是构建扎实的计算机科学技能的基石。以下是我在学习过程中的具
体收获。
1) 基本概念：掌握了计算机硬件、软件、操作系统、网络和编程语言等基础概
念。尤其是本次网页整体的搭建使用的三种技术：HTML、CSS 和 JavaScript。
2) 逻辑思维：通过了解算法和数据结构，提高了逻辑思维和问题解决能力。
3) 编程技能：了解了编程语言的内在逻辑，能够编写简单的程序和脚本。
4) 技术视野：对计算机科学的各个领域有了基本的了解，包括人工智能、数据
库、网络安全等。
5) 自学能力：在学习过程中，学会了如何合理利用网络资源和书籍自学新技
术，如何寻找所需资源，并对其做合理甄别。
并将所学知识应用于实际项目中，通过实践加深理解。
5.2
文档撰写工具 LaTeX
采用 LaTeX 进行文档编撰是一项对于追求卓越文档呈现的学者和专业人士
极具价值的技能，尤其适用于科学、数学以及技术文档的制作。
1) 专业排版：精通 LaTeX 意味着能够制作出具有出版级标准的文档，其排版
效果在学术出版领域备受推崇。
2) 数学表达：通过 LaTeX，可以高效且准确地录入包括高级数学符号和复杂
公式在内的各种数学表达式。
3) 文档组织：学习 LaTeX 有助于掌握如何有效地组织和构建大型和复杂的文
档，确保内容的逻辑性和条理性。
通过学习 LaTeX，用户将能够创建出既美观又功能强大的文档，这对于学术研究
和专业出版来说是极其重要的。

4
网页设计小结
在制作网页的过程中，由于对网页制作知识并不熟练，多次出现动画效果错
误的情况，而修改的过程也十分枯燥煎熬，因为大部分的错误不易察觉。
其中最突出的问题莫过于网页中的图片大小调节与位置问题。在首页下方
的图片其实一一开始会过于宽且“完美”地截掉人物的头，角色介绍界面的画框
会填充不满，以及图片有时放大后会过度模糊。
为了解决这类问题，我去专门找了一些教学视频学调整图片的大小与位置，
让最后的呈现效果有所好转。
另外就是画面的问题。最开始制作时网页的风格其实是纯黑白色，显得过于
死板。字体也是同样的黑色默认字体，重点不清晰，看起来令人眼花缭乱。
我于是想到先前学习的 wps 课程中可以修改字体，觉得网页制作也一定可
以。于是我上网搜索学习如何更改字体样式、大小与颜色等等，顺便学习到如何
改变背景色，由此改变了网页的整体样貌。
在制作动画效果时由于我并不熟练，出现了不少错误，但是感谢让人们分享
经验的互联网，我得以找出问题并且修正。

课程的收获和建议
感谢老师的课程讲解让我打牢了 html 网页制作的基础，让我对网页制作的
整体体系有了初步的认识。老师在课上也建议我们可以去一些教学网页系统学
习，并给出了推荐的平台，给我的网页设计提供了建议。
虽然用老师课上讲解的知识或许可以做一个简单的网页，但是并不能做出
我现在的成品。其实我并没有必要对于动画效果刻意追求，这也许只是一个结
课作业，但是我发自内心地希望我可以学习掌握更多知识，同时，我也真心喜爱
《汉密尔顿》这部优秀的作品。并希望我可以尽我所能把这个网页做得更好。
最后我发现，其实我并没有去花大量时间学习系统教学的网课，但是依旧
在紧跟实战教学视频中慢慢掌握很多小技巧。前端设计的很多前辈有的是经验，
对这些经验的汲取也是学习成长过程的一部分。

对计算机基础知识的学习，让我对计算机有了初步的系统的认识，比如数据

网页设计小结
作为一位之前从未接触过前端的“门外汉”，网页设计对我来说显然不是一
项容易的项目作业. 起初也曾因自己初试 html 与 css 不了解语法规则与调试操作
而感到焦虑。
然而，通过查询 bilibili、csdn 等平台进行学习，跟着视频中的老师们一步一
步的编译调试、一行一行的代码编写，逐渐搭建起完整的知识体系，在掌握最基
本的技术之后，我渐渐开始享受设计的过程，选择了自己喜欢与熟悉的主题，在
热爱的驱使下的输出让我成就感满满，同时信心也渐渐增长。
熟练掌握最基本的操作之后，我便不满足于单调的界面设计，于是我再次打
开各种论坛，这一次，我开始吸取更多创意十足的设计，并希望将这些美妙的设
计也应用到我自己的网站，于是，在这一过程中我进一步接触了更多更丰富的语
法，学习了编写更复杂多样的程序，也能够游刃有余地在我的网页中展示自我
了。
总而言之，我认为这是一次非常有意义的挑战，我在其中提升了技术，获得
了成就感，找到了热爱，可谓收获颇丰。

华 中 科 技 大 学 课 程 实 验 报 告
5
课程的收获和建议
通过新生实践课程的学习，我汲取了丰富而多样的知识，了解和运用了多种
技术。课程循序渐进，让我能在每次课程学习的积累中逐步丰富知识体系，为最
终完成网页设计的项目打下了坚实的基础。
我认为课程的重点可以更突出一些。新生实践课由多维的课程体系构成，然
而相对于如 Python、Git 等更进阶实用的课程内容的学习，一些过于基础的、实
用性和操纵性较低的学习内容也许可以简单带过，突出重点，而让我们将更多的
时间用于理解与实践，提高学习效率。
5.1
计算机基础知识
很多知识在我们平时日常使用计算机的过程中已经充分掌握，但仍有许多
值得关注的细节，仍有许多意外的收获。
同时，作为计算机系的学生了解计算机基础知识有利于我们更好地学习专
业知识发挥专业特长，为今后的学习打下基础。
5.2
文档撰写工具 LaTeX
学习使用 LaTeX 排版系统不但极大程度地丰富了排版技巧，也大幅提高了
文档编辑效率，让我意识到文档撰写除了最基础的 word 还有更多的新花样。同
时文档设计的基本逻辑也为网页设计项目做了准备。
5.3
编程工具 Python
对比正在学习的更为底层和基础的 C 语言，Python 语言更加简洁方便，易
于上手。入门 Python 编程语言，掌握了基本的语句，在头歌实践中熟练了程序
设计，内容密度、上手难度不大，但在实践中学习确实收获颇丰。
建议更注重编程实践，可适当减少理论讲解时间，而以实操演示和亲身实践
取而代之，在实践中学习编程语言比片面浅层的理论学习更实际与高效。
